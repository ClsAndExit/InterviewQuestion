常用排序算法类别：
插入排序：
    直接插入：（稳定）  时间复杂度(O(n^2))  最好（O(n)） 最坏（O(n^2)）
    shell排序：（不稳定）时间复杂度(O(n^1.3))  最好（O(n)） 最坏（O(n^2)）
选择排序：
    直接选择排序：（不稳定）  时间复杂度(O(n^2))  最好（O(n)） 最坏（O(n^2)）
    堆排序：（不稳定）
交换排序：
    冒泡排序：（稳定）  时间复杂度(O(n^2))  最好（O(n)） 最坏（O(n^2)）
    快速排序：（不稳定）  时间复杂度(O(nlog2n))  最好（O(nlog2n)） 最坏（O(n^2)）
归并排序：（稳定）
基数排序：（稳定）
